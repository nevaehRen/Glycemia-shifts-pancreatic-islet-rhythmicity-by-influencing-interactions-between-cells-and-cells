Glycemia shifts pancreatic islet rhythmicity by influencing interactions between δ cells and α cells (https://doi.org/10.1101/2025.08.09.669458) 

This file contains experimental data from 3 sets of IVGTT (Intravenous Glucose Tolerance Test) conducted on INS-GCAMP-wt/ob mice. The included data categories are:
1. Ca trace (Calcium signal trace)
2. Blood glucose concentrations
3. Oscillation peaks (of Ca signal/blood glucose)
